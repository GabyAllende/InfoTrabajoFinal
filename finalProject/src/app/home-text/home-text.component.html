Visualizador de Archivos CSV

Aplicación web que permite subir un archivo Excel (csv) y elegir las columnas de este para poder desplegar los gráficos estadísticos con la librería matplotlib.

Requerimientos:

  • Tener una página de inicio con la información del proyecto.
  • Habilitar un espacio para cargar un archivo csv
  • Poder elegir entre las opciones de gráficos scatter, lineales, barras, barrasY y Pie
  • Poder seleccionar las distintas columnas del archivo.
